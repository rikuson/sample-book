# CLAUDE.md

このファイルは Claude Code 固有のガイダンスを提供します。

**共通ルールは AGENTS.md を参照してください。**

## Claude Code 固有の設定

### 推奨ワークフロー

1. 原稿編集時は `npx textlint` でエラーを確認してから修正
2. 複数ファイルの一括編集は並列で実行可能
3. ビルド確認が必要な場合は `make epub` を実行

### Skills の活用

- `/commit` - 変更内容からコミットメッセージを生成
- `/plan` - タスクの実装計画を作成

### 注意事項

- 大量の変更を行う前にユーザーに確認を取る
- textlint エラーは可能な限り自動修正を試みる
- 表記ゆれは `prh.yml` の定義に従う
